What is robots.txt

Robots.txt is a file used to give instructions to web robots. it helps web owners to reduce the amount of web robots that scan the web if the owner doesnt want to, since web robots usually check the robot.txt file before it checks your website. This can be configured by adding a robots.txt file to the root folder of your project and that includes a "User-agent" that decides which robots the file apply to. This is followed by a "Disallow" in which one can determine which, if not all, parts of the website the web robot can visit.

Introduction to humans.txt

Humans.txt is basically a text with the information of the people who have contributed to create the website. This is done by making a humans.txt file in the root folder of the project and writing /* TEAM */ which are the participants in creating the website, /* THANKS */ which is a section for appriciation...etc.

Using Open Graph

Open graph is a technology that allows a user of a social media integrate with a website while in the social media. This lets developers choose what to show when their website is shared in a social media. Developer can add meta property of different kinds like url, Type, title... of a website inorder for a social media to show when the website is being shared.

Why different files fo css and html

First of all, having different files for the design and content can be more efficient and clearer and better structure. It is also easier for developers when writing diferent pages with the same style. Instead of having to write the design again and again, one can use the same file for several pages. Another advantage is that it allows browsers to cache the .css files which makes loading time faster.
